# Go后端开发学习笔记仓库

 ## 仓库简介 

本仓库用于日常沉淀 **后端开发** 全栈学习笔记，以 Markdown 形式整理归档，涵盖基础语法、框架使用、源码阅读、微服务、中间件、云原生、DevOps、工程化实践等各类技术知识点。 

持续更新学习过程中的知识点总结、踩坑记录、实战经验、代码示例与最佳实践，方便随时查阅复盘，也欢迎同行参考交流。 

## 内容涵盖范围 

- Go 基础语法、标准库、进阶特性与底层原理 

- Gin/Goframe/Echo 等主流 Web 框架实战 

- MySQL、Redis、MongoDB 等数据库使用与优化

- 微服务：GRPC、服务注册发现、配置中心、网关

- 消息队列：Kafka、RabbitMQ 实战与原理

- 缓存、分布式事务、限流、熔断、接口设计

- Docker、K8s、CI/CD 云原生部署运维 - 常用工具、开发规范、项目架构设计与踩坑记录。

## 使用说明 
1. 所有笔记均采用 Markdown 编写，结构清晰、可直接在线阅读； 
2. 按技术分类归档，方便按需查找对应知识点； 
3. 仓库长期持续更新，不断补充新的学习内容与实战案例；  
## 寄语 

日积月累，深耕后端；以笔记沉淀成长，用技术筑牢根基。
